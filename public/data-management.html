CSV Format Requirements

Each CSV file must include a header row. Required columns vary by data type:

  • Chase Reserve: Transaction Date*, Description*, Amount* (standard Chase CSV export)
  • Categories: Name*, Type* (income/expense/transfer)
  • Accounts: Account Name*, Type (checking/savings/credit_card)
  • Rules: Match Value*, Category*
  • Recurring: Description*, Amount*
  • Budgets: Amount*, Category (optional)

* Required field
